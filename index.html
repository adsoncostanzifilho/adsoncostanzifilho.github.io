Text Mining Tool

By Adson Costanzi Filho on November 24, 2020

Overview Text Mining Tool is an interface developed in R using mainly the shiny, tidytext and rtweet packages. The idea of this interface is to allow to you make your own text analysis using live Twitter data. The tool is online on shinyapps’ repository at the address https://adsoncostanzi.shinyapps.io/TextMining/. How does this interface work? The interface is divided into 5 tabs presented in the menu on the left: Home , Search , Word Cloud , Sentiment Analysis and, Topic Modeling .

soothsayeR

By Adson Costanzi Filho on November 23, 2020

Overview SoothsayeR is an interface based on the old prank called ‘Peter answers’.. The game’s idea is to play with your friends making questions that you know the answer, and make them believe that R is who is answering… The app demonstrates how easy it is to integrate Shiny with Javascript through the shinyjs library. The tool is online on shinyapps’ repository at the address https://adsoncostanzi.shinyapps.io/soothsayeR/.
